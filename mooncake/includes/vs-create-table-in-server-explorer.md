你可以通过使用 Visual Studio **服务器资源管理器**创建 Azure 存储表。

![服务器资源管理器表][Image1]

1. 在“视图”菜单中，单击“服务器资源管理器”。
2. 在服务器资源管理器，展开你的订阅的 **Azure** 节点，展开“存储”节点以及在 Azure 存储连接服务中指定的存储帐户的节点。
3. 选择“表”节点并从上下文菜单中选择“创建表”。
4. 输入表的名称，然后选择“确定”。
